# ansible-consul

Consul is a tool for service discovery and configuration. Consul is distributed, highly available, and extremely scalable.

Tunables
--------
* None

Dependencies
------------
* [colstrom.go](/colstrom/ansible-go)

Example Playbook
----------------
    - hosts: servers
      roles:
         - role: colstrom.consul
